# BOM-XML2EXCEL
## 説明
KicadのEeschemaで出力できるパーツリスト(BOM)をEXCELに変換するソフトです。

## インストール方法

[ここから実行ファイルを入手できます。](https://github.com/tofuso/bom-xml2excel/release)

## 使い方

```
bom-xml2excel {XMLのファイル名}
```

EXCELファイルはXMLのファイル名と同名になります。

```
bom-xml2excel {XMLのファイル名} --out {EXCELのファイル名}
```

EXCELファイルの名前を指定できます。
